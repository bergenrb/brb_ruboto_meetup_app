brb_ruboto_meetup_app
=====================

Sample Ruboto app showing very basic integration Meetup API.

First, get your Meetup API key from:
  http://www.meetup.com/meetup_api/key/

Enter the Meetup API key into file:
  res/values/strings.xml

To try it out, connect mobile device or start Android emulator, then
  rake install start

Any troubles, try running:
  logcat D

Built for android-10 (SDK v2.3.3), but might work down to android-8 (SDK v2.2).
